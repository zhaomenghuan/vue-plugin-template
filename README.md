# vue-plugin-template

一个基于 webpack 的 Vue.js 插件开发工程模板.

## 使用

安装 vue-cli：

```
$ npm install -g vue-cli
```

安装模板工程：
```
$ vue init zhaomenghuan/vue-plugin-template my-project
$ cd my-project
$ npm install
$ npm run dev
```

## 功能

```
npm run dev - 运行实例
npm run build - 生成实例代码包
npm run release - 生成插件库文件
npm run doc - 生成插件文档
```
